#### 1. 获取仓库列表

```
GET /data_api/v1/repositories
```

region_id 枚举

| region_id | 描述       |
| --------- | ---------- |
| 1         | 回民区     |
| 2         | 玉泉区     |
| 3         | 新城区     |
| 4         | 赛罕区     |
| 5         | 托克托县   |
| 6         | 清水河县   |
| 7         | 武川县     |
| 8         | 和林格尔县 |
| 9         | 土默特左旗 |

```
{
  err_code: 0,
  err_msg: string // 错误信息
  data: {
    items: [{
      id: int, // 仓库id
      name: string, // 仓库名称
      address: string, // 地址
      longitude: string, // 经度
      latitude: string, // 纬度
      region_id: int, // 区域id
      store_rate: string, // 存储率
    }],
  }
}
```
